# Idea 1

## Title: **Homesteader's Notebook**

## Overview
This website is designed to help make homesteading a bit easier.  For animals, it will provide a comprehensive list that will be filterable and provide links to breeders who can provide animals.  In addition, it has helpful tips and tricks to make homesteading easier like:
- Ways to preserve eggs for the winter

## Features
- Animal lookup
    - Filter based on a variety of restrictions
    - Item viewing
        - More details about the animal
        - links to places the animal may be purchased
        - Similar breeds
- Garden lookup
    - Filter based on a variety of restrictions
    - Item viewing
        - More details about the plant
        - links to places the plant may be purchased
        - Similar varieties
- Tips made easy

## Tech Stack

- Frontend
    - HTML
    - CSS
    - Javascript (Modular)
- Backend
    - Node.js
- Database
    - MySQL Server


## Process Description
This will be a fairly straight forward website. Its main purpose is to provide information.

There will be two header dropdowns for animals and garden plants.  The list that appears will allow the user to filter based on a variety of factors.
- heritage
- organic
- color (skin, fur, egg, etc)
- days to maturaty (when to harvest)
- Plant hardiness

Website will allow the user to lookup their hardiness zone based on the USDA lookup API and save it to their local settings.

When there is something that is interesting, the user can save for later or add it homestead.  In this way, a person can track what they have in their garden or when breeds of chickens they have in their coop.

Finding a provider of the animal or plant, a list of companies will be listed to make shopping easier.
