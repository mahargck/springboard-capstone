# Idea 2

## Title: **The Pantry**

## Overview
This makes planning easy for daily meals, freezer meals, etc.

Add a series of meals to the website and it will generate a shopping list.  If there are allergens, subsitutions may be available or variations.  Save recipies to a list and send them to a friend to help work on a recipe list for group activities.

## Features
Search site's recipes for items to make.  If one is not present, add recipe by just adding a link to an outside website.  Backend will search page for instructions and ingredients.

Add recipe to your list for a calendar menu, weekly menu, or custom menu for events or group cooking get-togethers.


## Tech Stack

- Frontend
    - HTML
    - CSS
    - Javascript (Modular)
- Backend
    - Node.js
- Database
    - MySQL Server


## Process Description
Search site for recipies or use the random lookup based on category:
- breakfast, lunch, entree, side, dessert
- holiday, gluten free, high probiotic, etc

or

- Recently viewed recipes

Click on a recipe to get **just** the ingredients and instructions.  Skip the story behind the recipe.
Add recipe to your menu; 'Save for Later', 'Calendar', 'Weekly', or 'Custom'.
Adjust your serving change the 1x multiplier from 1x to 2x, 3x, or other.
Click on your menu to get a shopping list.
